Course Units

Abstract Presentation of 5 Units

Unit 1: Introduction to NLP

Foundations and Language Processing Pipeline

Overview of Natural Language Processing

Natural Language Processing (NLP) is a branch of artificial intelligence that focuses on the interaction between computers and humans using natural language. The ultimate objective is to read, decipher, understand, and make sense of human languages in a manner that is valuable.

Key Goal: Enable machines to understand, interpret, and generate human language in a way that is both meaningful and useful.

Language Processing Pipeline

The NLP pipeline consists of several stages:

  1. Text Acquisition - Collecting raw text data
  2. Text Cleaning - Removing noise, HTML tags
  3. Tokenization - Breaking into words/sentences
  4. Normalization - Lowercase, stemming, lemmatization
  5. Feature Extraction - BoW, TF-IDF, embeddings
  6. Modeling - ML/DL for specific tasks

Challenges in NLP

  • Ambiguity - Lexical, syntactic, semantic
  • Context Dependency - Requires world knowledge
  • Coreference Resolution - Pronoun reference
  • Negation & Sarcasm - Non-literal meaning

Unit 2: Morphological Analysis & Text Processing

Word Formation, Stemming, and Lemmatization

Morphology and Morphemes

Morphology studies word structure. A morpheme is the smallest meaningful unit.

  • Free: cat, run
  • Bound: un-, -ed
  • Inflectional: cats, walked
  • Derivational: unhappy

Stemming Algorithms

Reduces words to root form by removing suffixes.

porter_stemmer.stem("running") → "run"
porter_stemmer.stem("happily") → "happi"

Lemmatization

Reduces to dictionary form using context and POS.

lemmatizer.lemmatize("better", pos='a') → "good"
Stemming: Fast, crude | Lemmatization: Accurate, linguistic

Unit 3: Syntax and Parsing

Phrase Structure, Dependency Parsing, and Grammar

Context-Free Grammar (CFG)

S → NP VP
NP → Det N
VP → V NP
Det → "the" | "a"

Parsing Techniques

  • Top-Down, Bottom-Up
  • CKY Algorithm
  • Earley Parser

Dependency Parsing

"The cat sat on the mat"
→ nsubj: cat → sat
→ prep: on → sat

Case Study

Text Summarization Using GPT-3

NLP Application

Text Summarization – Condenses long documents into concise, coherent summaries.

Downstream Task

Abstractive summarization using sequence-to-sequence generation with coherence and relevance.

Large Language Model

GPT-3 (175B parameters) by OpenAI – excels in few-shot learning for summarization.

Model Description

Autoregressive transformer decoder pre-trained on internet-scale text. Uses prompt engineering:

Prompt: "Summarize in 3 sentences: [article]"

Achieves ROUGE-L ~0.45 on CNN/Daily Mail. Strengths: fluency, zero-shot capability. Limitations: hallucinations, bias.

Conclusion

GPT-3 enables scalable, high-quality summarization with minimal fine-tuning. Ideal for journalism, research, and content curation. Future: domain-specific fine-tuning for precision.
